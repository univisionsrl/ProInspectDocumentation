PDF417
======

![](../../../../img/x_Graphics/Tools/UvfIdToolsStdPdf417-0.png)

Overview
--------

The PDF417 barcode tool decodes PDF417 stacked barcodes. PDF417 is a Â multirow, variable-length 2D symbology that offers high data capacity and error correction capability.

Results
-------

| --- | --- |
